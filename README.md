# Raylib Tetris

This is a Tetris clone implemented in C++ using the Raylib library.

## Controls

- **UP Arrow Key:** Rotate Block Clockwise
- **Z:** Rotate Block Counter-clockwise
- **Left/Right/Down Arrow Keys:** Move Block
- **Spacebar:** Hard Drop
- **C:** Hold Block
- **R:** Restart Game
- 
## Cheats

- **I**: Change Block
- **O:** Add Score
